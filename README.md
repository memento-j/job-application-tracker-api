# Job Applicaiton Tracker API

REST API to help track and manage my job applications while getting practice with Spring Boot Development.

## Technologies Used

[Maven 4.0](https://maven.apache.org/index.html)
[Java 21](https://www.oracle.com/java/technologies/downloads/#java21)

**Backend**
- Spring Boot
- JWT for stateless authentication
- BCrypt for password hashing

**Database**
- PostgreSQL
- Docker for local database development
